# GWizard SPI

A GWizard module that uses the Service Provider Interface pattern to load modules.


## Maven

```xml
	<dependency>
		<groupId>com.voodoodyne.gwizard</groupId>
		<artifactId>gwizard-spi</artifactId>
		<version>${gwizard.version}</version>
	</dependency>
```
